وارد کردن آخرین وضعیت

چون این برنامه endpoint برای خوندن وضعیت واقعی سرور نداره (فقط POST برای تغییر داریم)، اگه جای دیگه‌ای (مثل Postman یا یه ابزار دیگه) وضعیت رو عوض کردی، خروجی JSON رو اینجا کپی/پیست کن تا جدول با وضعیت واقعی هماهنگ شه. فرمت باید مثل نمونه‌ای باشه که قبلاً فرستادی (آرایه‌ای با bank، service، status، last_updated).
